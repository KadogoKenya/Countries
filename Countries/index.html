About This Project

Welcome to our world exploration project! This app allows you to learn about various countries across the globe. You can discover information about countries, including their flags, capitals, regions, and more. It pulls real-time data from the REST Countries API to provide accurate and up-to-date information about each country.

This project aims to provide an interactive way to explore world geography and is perfect for travelers, students, or anyone curious about the world!
